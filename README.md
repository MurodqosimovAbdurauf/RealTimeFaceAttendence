# 🎓 Student Registration with Real-Time Face Recognition

This mobile application helps **schools, universities, and other institutions** to register students or pupils using **real-time face scanning**.  
The scanned face is automatically added to the registration list.  
It is **fast, reliable, and secure**.  
The mobile application is also able to determine whether the face is **real or fake**, preventing fraud during registration.  

---

## 🚀 Features
- 📷 Real-time **face detection and recognition**
- ⚡ **Fast and reliable** student registration
- 🔒 Fake face (photo/video) detection
- 🏫 Suitable for **schools, universities, and educational institutions**
- 📊 Automatic storage and management of registration data

---

## 🛠️ Technologies Used
- **Frontend / Mobile App**: Flutter (Dart)  
- **Machine Learning**: TensorFlow
- **Database**: SQLite  
- **Development Tools**: Android Studio
